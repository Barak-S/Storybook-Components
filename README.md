## Requirements

- Node: 14.15.4
- Yarn: 1.22.10

## Development

Install packages:

```bash
yarn install
```

Run development server:

```bash
yarn start
```

Lint:

```bash
yarn lint
```

Test:

```bash
yarn test
```

## UI

This project uses [Material UI](https://material-ui.com/) lib as the basic for the UI.
